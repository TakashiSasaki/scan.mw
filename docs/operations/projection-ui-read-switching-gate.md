# Projection UI Read Switching Gate

This document describes the future UI read-switching gate only.

**CRITICAL SAFETY NOTES:**
- UI read switching is not authorized by any current local tool.
- UI read switching requires separate approval.
- No UI read switching implementation is included in this stride.

## Preconditions for UI Read Switching

Preconditions should include:
- clean broad-enough reconciliation evidence
- successful operation validation
- rollback policy
- monitoring plan
- staged rollout plan
- explicit route/component read-path change plan
